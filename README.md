# Piet
Working Piet (https://www.dangermouse.net/esoteric/piet.html) interpreter and stk to Piet compiler

# Installation
Uses pypng for reading and writing images.

# Run
Call
```bash
python cli <path_to_file.smpl> ([gen||stk||piet||debug],)*
```
